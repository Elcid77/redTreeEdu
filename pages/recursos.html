Recursos para Niños

Recurso Descripción Enlace
Scratch Scratch es un entorno de programación visual desarrollado por el MIT Media Lab. Está diseñado para que niños y adolescentes aprendan a programar de manera divertida y creativa. Utiliza bloques de código que se arrastran. Scratch
App Inventor App Inventor es una herramienta de desarrollo de aplicaciones móviles creada por Google y actualmente mantenida por el MIT. App Inventor
MakeCode Micro:bit es una placa de desarrollo de hardware diseñada para la educación en programación y electrónica. MakeCode es un entorno de programación visual desarrollado por Microsoft para programar el Micro:bit, aunque también admite otros dispositivos. MakeCode

Recursos Adolescentes y Adultos

Recurso Descripción Enlace
Visual Studio Code Visual Studio Code es un editor de código fuente desarrollado por Microsoft. Visual Studio
XAMPP XAMPP es un paquete de software gratuito y de código abierto que facilita la creación y gestión de entornos de desarrollo local para aplicaciones web. XAMPP
GIT Git es un sistema de control de versiones distribuido ampliamente utilizado para el desarrollo de software colaborativo. GIT
